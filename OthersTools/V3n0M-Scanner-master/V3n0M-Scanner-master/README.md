## Install note

Clone the repository:

```
$ git clone https://github.com/v3n0m-Scanner/V3n0M-Scanner.git
```

Then go inside:

```
$ cd V3n0M-Scanner/
```

Then install it:

```
$ python3 setup.py install --user
```
